# AWS-Amplify for svelte
[aws amplify](https://aws-amplify.github.io/docs/js/)을 이용하여 스벨트 프레임워크 상에서 aws의 요소를 사용할 수 있도록 구성해놓은 상용구 코드입니다.

# Prerequisites
- install @ams-amplifycli

# configure
```bash
npm install -g @aws-amplify/cli
amplify configure
amplify add <resource>
```

# getting started 
```bash
git clone <repo>
npm install
```
